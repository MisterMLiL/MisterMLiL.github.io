1 Сервер

1.1 Установка сервера

Скачать архив по ссылке: Архив

Разархивировать папку Rust_Server в любую папку на диске (желательно SSD диск), на которой будет находиться сервер.

В папке Rust_Server запустить файл .Update.bat, он запустит обновление/установки сервера на диск, он будет занимать 6-8 гигов.

Сервер установлен, теперь в папке Rust_Server будет находиться сервер Rust


1.2 Настройка сервера

Откройте папку Rust_Server. В папке .server находиться файл .Start.bat.

Скопируйте туда следующий текст

@echo off cls :srat echo Server starting... RustDedicated.exe -batchmode ^ +server.hostname "Local Server" ^ +server.identity "LocalServer" ^ +server.port 28015 ^ +server.maxplayers 1 ^ +server.seed 1111 ^ +server.worldsize 3000 ^ -logFile "output.txt" ^ +rcon.port 28016 +rcon.password "654465" ^ goto start
server.hostname - название сервера отображаемого при заходе на него
server.identity - идентификатор сервера
server.port - порт сервера (не менять)
server.maxplayers - максимальное количество игроков (можно ставить 1)
server.seed - сид генерации карты сервера
server.worldsize - размер карты сервера (по стандарту 4000)

По стандарту будет создана карта "Procedural Map". Для того чтобы изменить карту на другую добавьте следующую строку в файл перед -logFile:

+server.level map_name
server.level - название карты которая будет генерироваться на сервере

Запускать сервер нужно с файла .Start.bat.


1.3 Подключение к серверу

Чтобы подключиться к локальному серверу, заходим в Rust.
Открываем консоль на кнопку F1.

Вводим команду:

client.connect 127.0.0.1:28015

Далее должен начаться процесс подключения к локальному серверу.

2 Oxide

Oxide — это модификация для игры Rust, которая предлагает широкие возможности для серверного моддинга.

С помощью Oxide вы можете установить и настроить различные плагины, добавляющие новые функции и улучшения в игру.

2.1 Установка Oxide

Скачать архив по ссылке: Архив

Разархивируем в папку Rust_Server/.server.
Запускаем сервер с файла .Start.bat

После запуска сервера вводим следующую команду прямо в консоль:

oxide.version

Если выдается логическая информация, а не ошибка, то все установлено корректно.


2.2 Установка плагинов

Плагины можно найти на сайте uMod: Перейти

После скачивания плагина файл необходимо скинуть в папку Rust_Server/.server/oxide/plugins

Если плагин не начал работать на сервере и в консоль ничего не написалось - перезапустить сервер.

3 Команды для сервера

3.1 Команды локального сервера

restart <секунды> - перезапуск сервера
env.time <значение> - установить внутриигровое время
ownerid <steam id> - удалить администратора
noclip - полет
god - режим бессмертия

3.2 Команды oxide

3.2.1 Группы

oxide.version - узнать версия oxide и проверить работает ли он
oxide.show groups - посмотреть созданные группы
oxide.show group <название> - посмотреть игроков в группе
oxide.group add <название> - создать новую группу с <названием>
oxide.group remove <название> - удалить группу с <названием>
oxide.usergroup add <ник|id> <название> - добавить игрока в группу
oxide.usergroup remove <ник|id> <название> - удалить игрока из группы

3.2.2 Разрешения

oxide.grant group <название> permission - добавить permission группе <название>
oxide.revoke group <название> permission - удалить permission у группы
oxide.grant user <ник|id> permission - дать permission игроку
oxide.revoke user <ник|id> permission - удалить permission у игрока
oxide.show perms - список permission
oxide.show user <ник|id> - посмотреть список permission у игрока

3.2.3 Плагины

oxide.load <название> - загрузить плагин
oxide.unload <название> - выгрузить плагин
oxide.reload <название> - перезагрузить плагин

4.1 Команды сервера

4.2 Команды Oxide

Данные команды необходимо ввести в консоль одна за одной.

Данные команды создают группу players, и добавляют туда игрока с steamid 76561198871926612 (вставить свой SteamId).

oxide.group add players oxide.usergroup add 76561198871926612 players

4.2.1 Плагин Skins

Ссылка на плагин на сайте UMod

Данная команда добавлять группе players permission на использование команды skins.use.

Данная команда позволяет игрокам в группе players выбирать скины из списка.

oxide.grant group players skins.use

Файл Skins.json необходимо закинуть в папку Rust_Server/.server/oxide/config


4.2.2 Плагин RemoverTool

Ссылка на плагин на сайте UMod

Данная команда добавлять группе players permission на использование команды removertool.normal.

Данная команда позволяет игрокам в группе players пользоваться RemoverTool для удаления объектов или построек.

oxide.grant group players removertool.normal

Данная команда добавлять группе players permission на использование команды removertool.admin.

Данная команда позволяет игрокам в группе players пользоваться админским RemoverTool для удаления объектов или построек.
В этом режиме можно удалить любой объект.

oxide.grant group players removertool.admin

4.2.3 Плагин TimeOfDay

Ссылка на плагин на сайте UMod

Данная команда позволяет сделать длину ночи равную 3.

tod.nightlength 3

Данная команда позволяет сделать длину дня равным 40

tod.daylength 40
